# ハードウェアキット

部品は入手性と教材全体での再利用性を優先する。章ごとに新しいセンサを増やさない。

## 最小構成（第1部〜第9部の大半をカバー）

| 部品 | 数 | 用途 | 備考 |
|---|---|---|---|
| ESP32-C6-DevKitC-1 | 1 | 全章 | 基準ボード。WROOM-1(8MB flash)搭載 |
| USB-Cケーブル（データ通信対応） | 1 | 書き込み・電源 | 充電専用ケーブル不可 |
| ブレッドボード | 1 | 配線 | 400穴以上推奨 |
| LED（赤など） | 2〜3 | GPIO出力 | Vf約2V |
| 抵抗 330Ω | 3〜5 | LED電流制限 | 220〜470Ωで代用可 |
| 抵抗 10kΩ | 2〜3 | プルアップ/ダウン実験 | 内蔵プルでも学べるが実験用 |
| タクトスイッチ | 2 | GPIO入力 | ボードのBOOTボタン(GPIO9)でも代用可 |
| ジャンパ線（オス-オス） | 20本程度 | 配線 | |

## 追加構成（第7・8・11・12部で使用）

| 部品 | 用途 | 使う章 | 備考 |
|---|---|---|---|
| 可変抵抗 10kΩ | ADC・分圧 | 第7部 | 3端子 |
| I2C温湿度センサ（SHT30/SHT31等） | I2C | 第8部 | 3.3V品。教材のI2C章で共通使用 |
| SPIデバイス（例: 小型SPIディスプレイ） | SPI | 第8部 | 3.3V品 |
| サーボ（SG90等） | PWM | 第7部 | 外部5V電源推奨（後述の注意参照） |
| 2台目のESP32-C6-DevKitC-1 | ESP-NOW/最終プロジェクト | 第11・12部 | 送信側・受信側 |
| TWAIトランシーバ×2（TJA1051/3.3V対応品等） | TWAI | 第8部 | **必須。C6ピンをCANバスへ直結禁止** |
| 終端抵抗 120Ω×2 | TWAI | 第8部 | バス両端 |
| USB電流計または電流測定できるテスター | 消費電力測定 | 第12部 | µA測定はテスターのレンジに注意 |
| タクトスイッチ 4個（+1N4148ダイオード4本は任意） | キーマトリクス | 応用編 | 行=GPIO18/19、列=GPIO20/21 |

## 教材共通のピン割り当て

章をまたいで同じ用途には同じピンを使う（examplesと一致させる）。

| 用途 | ピン | 備考 |
|---|---|---|
| LED（メイン） | GPIO10 | 330Ω経由。全Lチカ・PWM章で共通 |
| LED2 / サーボ | GPIO11 | |
| ボタン | GPIO9 | オンボードBOOTボタン（内部プルアップ） |
| ADC（可変抵抗） | GPIO2 | ADC1_CH2 |
| I2C | SDA=GPIO6, SCL=GPIO7 | |
| UART1（ループバック実験） | TX=GPIO23, RX=GPIO22 | |
| SPI2 | SCK=GPIO19, MOSI=GPIO18, MISO=GPIO20, CS=GPIO21 | |
| TWAI | TX=GPIO2, RX=GPIO3 | トランシーバのTXD/RXDへ。ADC章と排他（配線し直し） |
| Deep Sleep GPIO起床 | GPIO7 | LP GPIO(0–7)のみDeep Sleep起床可。I2C章と排他 |

## 配線の共通注意

- ロジック電圧は**3.3V**。5V信号を直接GPIOへ入れない
- GPIOの絶対最大は3.6V。ピンあたりの駆動能力はソース約40mA・シンク約28mA（データシート典型値）だが、LEDは必ず抵抗で数mAに制限する
- サーボのモーター電源はUSBの5Vピンから取ると電圧降下でボードがリセットすることがある。別電源＋GND共通が安全
- 使用可能ピンと注意ピン（ストラッピング等）は docs/research/esp32c6-hardware.md を参照
